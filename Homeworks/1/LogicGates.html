A computer is built up of thousands of switches and those switches are built up of transistors

Logic Gate- An electronic circuit that performs a boolean function

Type of Gate Description
OR Gate If the output is true if either or both inputs are true
AND Gate The output is true if all inputs are true
NOT Gate The output is the inverse of the input
NAND Gate The output is true if any input is false
NOR Gate The output is true only when all inputs are false
XOR Gate The output is true if either input is true but not if both inputs are true
