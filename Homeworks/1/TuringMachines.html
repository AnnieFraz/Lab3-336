Turing Machines

Logic Gates Life Cycle Main


A Turing Machine is an FSM(Finite State Machine) that controls one or more tapes, where at least one tape is of unbounded length (i.e. infinitely long).

Alan Turing used his Turing machine to show that it is not possible to determine if any given problem has a solution, or not. This has serious implications in computing. It means that it may not always be possible to determine if a particular problem is unsolvable, or if it has just not been solved yet.



Turing Machines consist of:

  1. An input alphabet
  2. A tape that goes on infinitely in one direction
  3. An output alphabet(characters that can be printed onto the tape)
  4. A tape head that is used to read the contents of the tape one cell at a time
  5. A finite set of states, including one initial state
  6. A program (FSM set of instructions) that tells it which state to change to, which direction to move the tape head and which member of the output alphabet to place on the tape – based on the current state and the input received.
