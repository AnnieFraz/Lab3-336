There are 6 steps to the software development cycle:

  1. Planning
    • What is the project?
    • How will it help the user>
  2. Analysis

      3. This is where you create programming objectives by using SMART

    • Specific - Come up with an unammbigious idea
    • Measurable - How can you prove that you met this objective
    • Attainable - Do you have the sources and resources avaliable to complete the project
    • Relevant - How will the project meet the user's needs?
    • Time - Can you complete the project in the time frame?>
  3. Design
    • Here is where you start designing the components
    • It lets you think how things are designed and hopefully avoid mistakes
  4. Implementation

    5. This is where you do these actions:

    • Programming the software
    • Installation of Hardware/Software
    • Creation of data files
    • User's training
    • System documention creation
  5. Testing

    6. There are different types of testing

    • Black box
    • White box
    • Dry Run
  6. Maintenance

    7. After the project has been completed, does it need updating?
